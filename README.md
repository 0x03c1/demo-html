# Webpage

Projeto simples de página estática.

## Estrutura

- `index.html`: Página principal.
- `css/style.css`: Estilos da página.

## Pré-requisitos

- Navegador moderno (Chrome, Firefox, Edge, Safari).

## Como executar

Abra o arquivo `index.html` diretamente no navegador ou use um servidor local:

## Desenvolvimento

- Edite `css/style.css` para ajustes visuais.
- Mantenha HTML semântico em `index.html`.
- Arquivos ignorados pelo Git estão em `.gitignore`.

## Publicação

- Hospede em qualquer serviço de páginas estáticas (GitHub Pages, Netlify, Vercel).
- Garanta que o caminho do CSS em `index.html` esteja correto.
